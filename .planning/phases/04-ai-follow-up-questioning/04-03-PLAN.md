**1. Create `lib/features/ai/presentation/skill_chips.dart`:**

A `StatelessWidget` that renders a horizontal Wrap of colored Chip widgets for skill tags.

Props:
- `List<String> tags` -- the skill tag strings
- `bool allowDismiss` -- if true, shows delete icon on each chip (for detail screen); if false, read-only (for card)
- `ValueChanged<String>? onDismiss` -- callback when a tag is dismissed
- `bool compact` -- if true, uses smaller chips suitable for card display

Build:
- `Wrap` with `spacing: 6`, `runSpacing: 4`
- Each tag is a `Chip` (or `InputChip` when `allowDismiss` is true):
  - Label: tag text in `labelSmall` (compact) or `labelMedium`
  - Background: Use a deterministic color from the tag string hash. Create a helper `Color _tagColor(String tag, ColorScheme cs)` that picks from a palette of 8 M3-friendly colors: `cs.primaryContainer`, `cs.secondaryContainer`, `cs.tertiaryContainer`, and 5 more generated from different hue shifts of the seed. Use `tag.hashCode % 8` to pick.
  - Delete icon: `Icons.close`, size 14, only when `allowDismiss` is true
  - `onDeleted`: calls `onDismiss?.call(tag)`
  - Shape: `StadiumBorder()` with `side: BorderSide.none`
  - `materialTapTargetSize: MaterialTapTargetSize.shrinkWrap` for compact mode
  - `visualDensity: VisualDensity.compact` for compact mode
  - `padding: EdgeInsets.symmetric(horizontal: 4)` for compact mode

**2. Create `lib/features/ai/presentation/impact_indicator.dart`:**

A `StatelessWidget` that renders a small colored dot indicating impact level.

Props:
- `String? impactLevel` -- the impact level string ('low', 'medium', 'high', 'critical')

Build:
- If `impactLevel` is null, return `SizedBox.shrink()`.
- Render a `Container` with `width: 8, height: 8`, circular `BoxDecoration`.
- Color mapping:
  - `'low'`: `colorScheme.outline` (subtle grey)
  - `'medium'`: `colorScheme.tertiary`
  - `'high'`: `colorScheme.primary`
  - `'critical'`: `colorScheme.error`
- Wrap in a `Tooltip` showing the impact level (e.g., "Impact: High").

**3. Create `lib/features/ai/presentation/category_badge.dart`:**

Props:

- `List<String>? secondaryCategories` -- secondary categories (shown only in detail mode, not card)
- `bool showSecondary` -- whether to show secondary categories (default false)
- `VoidCallback? onTap` -- if provided, makes the badge tappable (for override UI)

Build:
- If `primaryCategory` is null, return `SizedBox.shrink()`.

- Primary: A `Text` widget with `labelSmall` style, `colorScheme.onSurfaceVariant` color, `fontWeight: FontWeight.w600`.
- If `showSecondary` and secondaryCategories is non-empty: show secondary categories as smaller text after the primary, separated by a dot.
- If `onTap` is provided: wrap in `InkWell` with a subtle dropdown icon to indicate editability.
