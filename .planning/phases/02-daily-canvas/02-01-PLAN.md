**1. Update convex/schema.ts** ŌĆö Add the entries table alongside the existing users table:

```typescript
entries: defineTable({
  userId: v.string(),        // Google ID of the owning user (matches AuthStateAuthenticated.userId)
  body: v.string(),          // Entry text content
  inputMethod: v.string(),   // "text" | "voice" ŌĆö for mic badge display on voice entries
})
  .index("by_user_creation", ["userId"])  // Filter today's entries by user; _creationTime auto-appended
  .searchIndex("search_body", {
    searchField: "body",
    filterFields: ["userId"],             // Scope full-text search to current user
  }),
```

Key decisions:
- `userId` is `v.string()` (NOT `v.id("users")`) ŌĆö matches existing auth pattern where `AuthStateAuthenticated.userId` can be Convex doc ID or Google ID fallback
- No explicit timestamp field ŌĆö `_creationTime` is auto-set by Convex (millisecond Unix timestamp), satisfying CANV-07
- `inputMethod` tracks "text" vs "voice" for the mic badge on voice entries (CANV-05)
- `by_user_creation` index on `["userId"]` ŌĆö Convex auto-appends `_creationTime` as the last column, giving us user-scoped time-ordered queries

**3. Create lib/features/today/domain/entry.dart** ŌĆö Entry data model:

```dart
class Entry {
  final String id;           // Convex _id
  final String userId;
  final String body;
  final String inputMethod;  // "text" | "voice"
  final int creationTime;    // Convex _creationTime (Unix ms)

  const Entry({
    required this.id,
    required this.userId,
    required this.body,
    required this.inputMethod,
    required this.creationTime,
  });

  factory Entry.fromJson(Map<String, dynamic> json) {
    return Entry(
      id: json['_id'] as String,
      userId: json['userId'] as String,
      body: json['body'] as String,
      inputMethod: json['inputMethod'] as String,
      creationTime: (json['_creationTime'] as num).toInt(),
    );
  }

  DateTime get createdAt => DateTime.fromMillisecondsSinceEpoch(creationTime);
  bool get isVoice => inputMethod == 'voice';
}
```

final currentUserIdProvider = Provider<String?>((ref) {

Create `todayEntriesProvider` ŌĆö a `StreamProvider.autoDispose<List<Entry>>` that:
